<h2>Installation</h2>

  URLer Repository requires <a href="https://www.python.org/">Python </a> v3 to run.
  
  <h3>Linux</h3>
  
  ```
  $ git clone https://github.com/rebl0x3r/URLer.git
  
  $ cd URLer

  $ chmod +x url.py
  $ pip3 install -r requirements.txt
   
  $ ./url.py
  ```
  Or just type :
  ```
  $ python3 url.py
   
  ```
